## 📖 About

This repository contains the **Statistics with R Programming Laboratory** experiments implemented using the **R Programming Language**. The experiments cover the fundamentals of R programming, statistical data analysis, data visualization, exploratory data analysis (EDA), regression models, and classification techniques using built-in datasets.

Each experiment includes:
- ✅ Question
- ✅ Aim
- ✅ Procedure
- ✅ R Code
- ✅ Input
- ✅ Output
- ✅ Result
